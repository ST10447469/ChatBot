This project is a simple console-based chatbot developed in C# that helps users learn about basic cybersecurity practices. The chatbot interacts with the user through a conversation-style interface and provides advice on topics such as password safety, phishing scams, suspicious links, and safe browsing habits.The idea behind this project is to simulate real-life situations where users might be exposed to cyber threats and guide them on how to respond safely. It’s designed to be beginner-friendly, both for users interacting with the bot and for developers learning how to build interactive applications.
Type the following into the chatbot to test different responses:
how are you?
what is your purpose?
what can i ask you?
tell me about passwords?
what is phishing?
how do I browse safely?
is this link safe?
exit
