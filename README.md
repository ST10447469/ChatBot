This project is a C#-based cybersecurity awareness chatbot developed to help users learn about important online safety practices through interactive conversations. The chatbot communicates with the user in a conversation-style format and provides guidance on topics such as password safety, phishing scams, suspicious links, privacy, and safe browsing habits.

The purpose of the project is to simulate real-life cybersecurity situations where users may encounter online threats and teach them how to respond safely and responsibly. The chatbot is designed to be beginner-friendly for both users and developers while also demonstrating important programming concepts such as Object-Oriented Programming (OOP), input validation, multimedia integration, dynamic responses, memory handling, and GUI development using WPF.

As the project progresses, the chatbot continues evolving from a simple console application into a more advanced cybersecurity assistant with improved conversation flow, sentiment detection, and dynamic response handling.
